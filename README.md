# GedcomGraph Canvas

GedcomGraph Canvas is a minimal graphical implementation of the Java library [GedcomGraph](https://github.com/michelesalvador/GedcomGraph).

GedcomGraph Canvas is a Gradle project written with Eclipse, and realized with Java AWT/Swing.

GedcomGraph Canvas can open a GEDCOM file (*.ged) and generate a graphic browsable output of a genealogical tree.
